#Phương thức trong HTTP:
__________________________________________________
| Method | Chức năng         | Thay đổi dữ liệu  |
| ------ | ----------------- | ----------------- |
| GET    | Lấy dữ liệu       | ❌ Không          |
| POST   | Tạo mới           | ✅ Có             |
| PUT    | Cập nhật toàn bộ  | ✅ Có             |
| PATCH  | Cập nhật một phần | ✅ Có             |
| DELETE | Xóa               | ✅ Có             |
--------------------------------------------------
